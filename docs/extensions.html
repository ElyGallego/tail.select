AJAX Plugin

Asynchronous Loader

The AJAX select Plugin allows you to load your set of select <option> elements in an asynchronous way to speed up your website or web application.

The AJAX functionallity can also only be triggered, when a search is applied. This is a perfect way to search through hundreds or thousands of available options without showing them all at once.

Columns and Collapse

The Columns select Plugin allows you to show multiple select option elements or optgroups side-by-side. Sometimes that doesn't just look better, it also improves to user experience.

Another functionallity, offered by this Plugin, is the possibility to collapse the single optgroups and the ungrouped options if grouplessName is set.

Input Plugin

Input and Create

The Input select Plugin turns the main label element into an editable input field, which can also be used by the user to create new options themselves (if the create option is enabled, of course).

This plugin can perfectly be combined with the movement plugin below, which also "moves" the selected options into the label element. Check out the advanced examples for more informations.

Move like a Pro

The Movement select Plugin allows you to move selected options to custom containers or to the main select label element. This can be perfectly used for tags or eMail receiver / cc / bcc field.

Another function of this plugin is to pin selected options on the top of the dropdown field, which allows a way easier ser experience.
