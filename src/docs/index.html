Documentation

Welcome to the tail.select documentation site. The following pages will show you how to use, extend and theme the complete environment, including tips and tricks the complete way along. First of all, you should check out the Installation and Usage page, if you're completely new to tail.select, otherwise you shouldn't miss the Migration page to learn how you can port your 0.5.x installation to the new release.

The tail.select script has some peculiarities, that you can use to your advantage. The Tricks & Good-to-Know page lists these and contains also some other neat tips for you.
